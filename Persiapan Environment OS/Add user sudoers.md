# Menambahkan user sudoers

## Add user `orion`
Gunakan perintah berikut untuk menambahkan user baru di linux:
```ssh
sudo adduser <namauser>
```
Untuk pindah user gunakan perintah:
```ssh
sudo su - <namauser>
```
Untuk menampilkan pengguna saat ini, gunakan perintah:
```ssh
whoami
```
